# Pingara

This is the `.github` repository for the **Pingara** organization.

## Purpose

This special repository contains configuration files and documentation that appear on Pingara's public organization profile on GitHub.

Files in this repository can include:
- `profile/README.md` - Content displayed on the organization's profile page
- Community health files (CODE_OF_CONDUCT.md, CONTRIBUTING.md, etc.)
- Default configuration files that apply across all repositories in the organization

Visit our organization profile at: https://github.com/Pingara

Visit our website at: https://www.pingara.io
